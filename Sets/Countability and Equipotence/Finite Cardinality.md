# Definition
Let $N_{p} = \set{n \in \mathbb{N} : n \leq p}$ for some $p \in \mathbb{N}$. A [[Set|set]] $M$ is ___finite___ if there exists a $p$ such that $M \sim N_p$[^1]

In this we say $M$ is of [[Cardinality|cardinality]] $p$ and we write $|M| = p$
